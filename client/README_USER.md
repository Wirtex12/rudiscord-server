# Voxit Messenger - Инструкция пользователя

## Установка

1. Скачайте файл `Voxit-Setup-X.X.X.exe` из Releases на GitHub
2. Запустите установщик
3. Нажмите "Next" → "Install" → "Finish"

## Первый запуск

1. Запустите Voxit из меню Пуск или с рабочего стола
2. Нажмите **"Создать аккаунт"**
3. Введите:
   - **Username** (уникальное имя, 3-20 символов)
   - **Email** (действующий email)
   - **Пароль** (минимум 6 символов)
4. Нажмите **"Зарегистрироваться"**

## Вход

1. Введите username и пароль
2. Нажмите **"Войти"**
3. Готово! Вы в приложении

## Профиль

### Изменение имени
1. Нажмите на аватар (внизу слева)
2. Нажмите ✏️ рядом с именем
3. Введите новое имя
4. Нажмите **"Save"**

### Загрузка аватара
1. Нажмите на аватар
2. Нажмите **"📷 Click to upload"**
3. Выберите изображение (JPG, PNG, GIF, WebP)
4. Готово!

### Удаление аватара
1. Нажмите на аватар
2. Нажмите **"Remove Avatar"**

### Удаление аккаунта
1. Нажмите на аватар
2. Прокрутите вниз
3. Нажмите **"Delete Account"** (красная кнопка)
4. Подтвердите удаление

⚠️ **Внимание:** Это действие необратимо!

## Друзья

### Добавление друга
1. Откройте **Settings** (клик на аватар)
2. Найдите секцию **"Add Friend"**
3. Введите **User ID** друга (формат: `1234-5678`)
4. Нажмите **"Send Request"**

### Принятие запроса
1. Откройте вкладку **"Pending"** в Friends Modal
2. Нажмите **✓** для принятия
3. Друг появится в списке друзей

### Отклонение запроса
1. Откройте вкладку **"Pending"**
2. Нажмите **✕** для отклонения

### Удаление друга
1. Откройте список друзей
2. Нажмите на друга
3. Нажмите **"Remove Friend"**

## Звонки

### Видеозвонок
1. Откройте чат с другом
2. Нажмите кнопку **📹** (видео)
3. Разрешите доступ к камере и микрофону
4. Дождитесь ответа
5. Готово!

### Аудиозвонок
1. Откройте чат с другом
2. Нажмите кнопку **📞** (аудио)
3. Разрешите доступ к микрофону
4. Готово!

### Завершение звонка
- Нажмите красную кнопку **📞**

### Отклонение звонка
- Нажмите красную кнопку **✕**

## Сообщения

### Отправка сообщения
1. Выберите друга в списке
2. Напишите сообщение в поле ввода
3. Нажмите **Enter** или кнопку **➤**

### Чтение сообщений
- ✅ Одно прочтение — сообщение доставлено
- ✅✅ Два прочтения — друг прочитал

## Настройки

### Проверка обновлений
1. Нажмите на аватар
2. Нажмите **"🔍 Check for Updates"**
3. Если есть обновление — оно загрузится автоматически

### Выход
1. Нажмите на аватар
2. Нажмите **"Logout"**

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `F12` | Открыть DevTools (отладка) |
| `Ctrl+R` | Обновить приложение |
| `Esc` | Закрыть модальное окно |

## Поддержка

Если что-то не работает:

1. **Проверьте интернет-соединение**
2. **Обновите приложение** (автообновление)
3. **Перезапустите приложение**
4. **Очистите кэш:**
   - Выйдите из аккаунта
   - Удалите приложение
   - Установите заново
